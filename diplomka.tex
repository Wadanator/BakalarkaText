\abstract{ % abstrakt
    Modulárny riadiaci systém pre interaktívne múzejné expozície bol navrhnutý
    a realizovaný s využitím platforiem Raspberry Pi a ESP32. Architektúra systému je postavená na princípoch internetu vecí (IoT),
    kde centrálna jednotka Raspberry Pi zabezpečuje orchestráciu scén,
    synchronizáciu audiovizuálneho obsahu a celkovú riadiacu logiku.
    Distribuované uzly na báze mikrokontrolérov ESP32 slúžia na priame ovládanie
    periférií, akými sú jednosmerné motory, relé moduly a vstupné senzory.
    Pre spoľahlivú a asynchrónnu komunikáciu medzi zariadeniami bol implementovaný
    protokol MQTT s dôrazom na sledovanie spätnej väzby a automatické zotavenie
    pri výpadkoch siete. Súčasťou riešenia je aj webový dashboard, ktorý
    operátorom múzea umožňuje monitorovať stav hardvéru, konfigurovať parametre
    a diagnostikovať systém v reálnom čase. Výsledkom práce je plne funkčný
    a nasadený riadiaci systém integrovaný do reálneho exponátu,
    ktorý demonštruje škálovateľnosť a nenáročnú údržbu
    v podmienkach každodennej múzejnej prevádzky.
}{ % abstrakt anglicky
    A modular control system for interactive museum exhibits was designed and
    implemented using Raspberry Pi and ESP32 platforms. The system architecture is built on Internet of Things (IoT)
    principles, where a central Raspberry Pi unit handles scene orchestration,
    audiovisual content synchronization, and overall control logic. Distributed
    nodes based on ESP32 microcontrollers are used to directly control peripherals
    such as DC motors, relay modules, and input sensors. For reliable and
    asynchronous communication between devices, the MQTT protocol was implemented,
    with an emphasis on feedback tracking and automatic recovery during network
    outages. The solution also includes a web dashboard that allows museum
    operators to monitor hardware status, configure parameters, and diagnose the
    system in real time. The result of the thesis is a fully functional and
    deployed control system integrated into a real exhibit,
    demonstrating scalability, and low maintenance requirements
    under the conditions of daily museum operation.
}{ % klicova slova
    interaktívna expozícia, riadiaci systém, Raspberry Pi, ESP32, MQTT, stavový automat, orchestrácia scén, webový dashboard, React
