\chapter{Zoznam použitých skratiek a symbolov}

\begin{description}[leftmargin=3.5cm, labelwidth=3.5cm, labelsep=1em, font=\bfseries]
\item[AC]     Alternating Current -- striedavý prúd
\item[CNC]    Computer Numerical Control -- počítačom riadené obrábanie
\item[DC]     Direct Current -- jednosmerný prúd
\item[DMX512] Digital Multiplex 512 -- štandard pre ovládanie javiskového osvetlenia a~efektov
\item[ESP32]  Označenie rodiny mikrokontrolérov od spoločnosti Espressif Systems
\item[FSM]    Finite State Machine -- konečný stavový automat
\item[GPIO]   General Purpose Input/Output -- vstupno-výstupné piny všeobecného použitia
\item[HDMI]   High-Definition Multimedia Interface -- multimediálne rozhranie
\item[HTTP]   HyperText Transfer Protocol -- protokol pre prenos hypertextu
\item[I2C]    Inter-Integrated Circuit -- dvojvodičová sériová zbernica
\item[IoT]    Internet of Things -- Internet vecí
\item[IPC]    Inter-Process Communication -- medziprocesová komunikácia
\item[JSON]   JavaScript Object Notation -- textový dátový formát
\item[LAN]    Local Area Network -- lokálna počítačová sieť
\item[LED]    Light Emitting Diode -- svetlo emitujúca dióda
\item[LWT]    Last Will and Testament -- mechanizmus poslednej vôle v~protokole MQTT
\item[MQTT]   Message Queuing Telemetry Transport -- protokol správ pre IoT
\item[OSC]    Open Sound Control -- sieťový protokol pre komunikáciu medzi multimediálnymi zariadeniami
\item[PLC]    Programmable Logic Controller -- programovateľný logický automat
\item[PoE]    Power over Ethernet -- napájanie cez Ethernet kábel
\item[PWM]    Pulse Width Modulation -- pulzne šírková modulácia
\item[QoS]    Quality of Service -- kvalita služby (úroveň spoľahlivosti doručenia správ)
\item[RAM]    Random Access Memory -- operačná pamäť
\item[RPi]    Raspberry Pi -- jednodoskový počítač od nadácie Raspberry Pi Foundation
\item[SBC]    Single Board Computer -- jednodoskový počítač
\item[SMPTE]  Society of Motion Picture and Television Engineers -- organizácia definujúca štandardy pre filmový a~televízny priemysel
\item[SPA]    Single Page Application -- jednostránková webová aplikácia
\item[SSR]    Solid State Relay -- polovodičové relé
\item[TCP/IP] Transmission Control Protocol / Internet Protocol -- sieťové protokoly
\item[Wi-Fi]  Wireless Fidelity -- bezdrôtová lokálna sieť (IEEE~802.11)
\item[XML]    eXtensible Markup Language -- rozšíriteľný značkovací jazyk
\item[YAML]   YAML Ain't Markup Language -- formát pre konfiguračné súbory
\end{description}
