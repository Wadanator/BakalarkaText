Prilohy k bakalarskej praci
===========================

Priloha A - Evidencia vyuzitia generativnej umelej inteligencie
  Evidencia je uvedena priamo v textovej casti prace.

Priloha B - Zdrojovy kod systemu
  Umiestnenie: Zdrojovy_kod/museum-system/

  Hlavne casti:
  - esp32/        Firmware pre ESP32 uzly
  - frontend/     Zdrojovy kod weboveho dashboardu
  - raspberry_pi/ Backend riadiaceho systemu pre Raspberry Pi

Priloha C - Dokumentacia instalacie
  Umiestnenie: Obrazky/ a Video/

  Hlavne casti:
  - Fyzicka_instalacia/ Fotografie fyzickeho exponatu a instalacie
  - Web_UI/             Screenshoty weboveho rozhrania
  - Video/              Videozaznam synchronizacie multimedii a fyzickych efektov

Poznamka:
  Zdrojovy kod bol ocisteny od git historie, docasnych suborov a cache suborov.
  Konfiguracne udaje su urcene na demonstraciu struktury systemu.
